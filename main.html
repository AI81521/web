시작하려면 클릭!

사용 방법
    1. setting버튼을 눌러 원하는 단어와 뜻을 입력한다.
    2. start버튼을 눌러 시작한다.
    3. 뜻 입력후 check버튼 누른다.
    4. 맞으면 correct 아니면 incorrect와 답이 나온다.
    5. next버튼을 눌러서 다시 3~4의 과정을 반복한다. 6. end버튼이 나오면 눌러 맞은 개수를 확인한다.
주의사항 설정할 때 처음에 단어 개수를 입력할 때 숫자를 입력해주세요.
오류가 나면 재시작을 하세요.
답 입력시 뜻과 완전히 동일하게 입력해야합니다.(','인식 안됨.)
